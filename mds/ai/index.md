# Bmob + AI：让你的 IDE 直接连接真实后端能力

在 Cursor 等支持 MCP 的 AI 工具中，直接完成建表、查数、生成代码与业务落地，不再只停留在“示例代码”。

## 为什么选择 Bmob AI 开发方案

- 连上即用：3 分钟接入 Bmob MCP，马上读到真实表结构
- 产出可用：结合 Skills 自动路由到正确平台（Web / Android / iOS / REST）
- 闭环开发：从需求描述到代码实现，再到数据验证一次完成

## 快速开始

- 立即配置 MCP：[`MCP 使用文档`](mcp/index.md)
- 查看 Skills 用法：[`Skills 使用文档`](skills/index.md)

## 你可以直接实现什么

- 用自然语言完成建表、写入、更新、删除、查询等数据操作
- 生成 Vue/React/Android/iOS 等多端可运行代码
- 按真实业务表结构生成类型定义、模型类与接口调用代码
- 遇到报错时，结合错误码与技能路由快速定位问题

## 推荐使用路径

1. 先在 MCP 文档中完成配置与连通验证
2. 在 Skills 文档中确认你的平台和任务路由
3. 按“先查真实表结构，再产出代码”的流程执行
